# Internet cafe time
We will help you save your money while using Internet cafes
# Run the program in Python IDE

`git clone https://github.com/ayadseghairi/Internet_cafe_time.py.git`

`pip install psutil`

`cd Internet_cafe_time.py`

`python3 main.py`
